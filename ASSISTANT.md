## 🧠 What is OpenClaw?

OpenClaw is an open-source platform for deploying isolated, remote‑manageable AI agent servers on your local network. It turns an old laptop, a Raspberry Pi, or any always‑on machine into a personal AI assistant you can chat with from your phone or computer.

**Highlights:**
- **Local-first:** Your data stays on your hardware.
- **Extensible:** Add skills (email, calendar, GitHub, IoT) on demand.
- **Persistent:** The assistant remembers across sessions and can run automated background tasks.

## 🤖 How to Get Your Own Assistant

1. **Grab the guide** — this repo contains the step‑by‑step setup for Ubuntu.
2. **Install OpenClaw** on your target machine (even an old PC works great).
3. **Chat** via the built‑in web UI or CLI.
4. **Customize** with skills like `github`, `email`, `weather`, and more.

Quick install (on a supported Linux machine):

```bash
npm install -g openclaw
openclaw onboard
```

Then open `http://localhost:18789` in your browser and start chatting.
